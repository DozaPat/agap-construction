# AGAP Construction Management System

A full-stack MERN (MongoDB, Express, React, Node.js) application for managing construction projects, workers, materials, tools, and expenses.

## Features
- User Authentication (Login/Logout)
- Dashboard with statistics and charts
- Project Management
- Worker Management
- Materials & Tools Inventory
- Expense Tracking
- Task Progress Monitoring
- PDF Report Generation

## Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS + Recharts
- **Backend**: Node.js + Express + MongoDB
- **Others**: JWT Auth, Zustand, jsPDF

## How to Run

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (new terminal)
cd frontend
npm install
npm run dev
